# Proyecto Cyberpunk Neon

Una página web estática de una sola página que explora el género cyberpunk a través de un manifiesto poético y un resumen histórico, todo envuelto en una estética de neón y alta tecnología.

## Características

- **Estética Cyberpunk:** Tema oscuro con una paleta de colores cian y magenta, inspirada en las luces de neón.
- **Tipografía Inmersiva:** Utiliza la fuente monoespaciada `Share Tech Mono` de Google Fonts para un toque tecnológico.
- **Animaciones CSS Puras:**
  - Efecto de parpadeo (flicker) en el título principal para simular un letrero de neón.
  - Efecto de "glitch" que añade una capa de distorsión digital.
- **Contenido Dual:**
  - Un **manifiesto poético** que sirve como introducción artística.
  - Un **resumen histórico** sobre el origen y la evolución del género cyberpunk, con una tabla de contenido para la navegación.
- **Diseño Ergonómico:** El texto de lectura principal ha sido ajustado para mejorar la legibilidad sin sacrificar el estilo.
- **Estructura Limpia:** Código HTML5 semántico y estilos CSS separados para un fácil mantenimiento.

## Estructura del Proyecto

```
CyberPunkProject/
├── CyberPunk.html  # Archivo principal con el contenido
└── style.css       # Hoja de estilos con todo el diseño y animaciones
```

## Cómo Empezar

No se requiere instalación. Simplemente clona o descarga los archivos y abre `CyberPunk.html` en tu navegador web preferido.

## Tecnologías Utilizadas

- HTML5
- CSS3 (Variables, Animaciones, Pseudo-elementos)
